Quick Start

Get started in 30 seconds

curl 
# Search for Python jobs in Bangalore
curl "http://localhost:8000/api/v1/jobs?q=python&location=Bangalore&limit=5"

# Match a resume against all jobs
curl -X POST "http://localhost:8000/api/v1/agent/search" \
  -H "Content-Type: application/json" \
  -d '{"skills":["python","sql","ml"],"experience_years":3,"preferred_locations":["Hyderabad"]}'
